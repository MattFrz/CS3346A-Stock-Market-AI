# Stock Market AI

## Dependencies
- Python 3.12.X

## Build Instructions

### 1. Clone Repository
- HTTPS URL:
```shell
https://github.com/MattFrz/CS3346A-Stock-Market-AI.git
```

### 2. Install Requirements
- Run this command in the project's root folder (...\CS3346A-Stock-Market-AI\\)
```shell
pip install -r requirements.txt
```

## Execution Pipeline
- The .py files should be run in the following order
```shell
data_collector.py -> data_preprocessor.py -> train.py -> evaluate.py -> predict.py
```
